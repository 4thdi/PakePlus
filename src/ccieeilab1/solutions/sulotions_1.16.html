1.16 Enabling Internet Access for FABD2

R12
router bgp 65001
neighbor 200.99.12.1 remote-as 19999
address-family ipv4
neighbor 200.99.12.1 activate

router ospf 1
default-information originate metric 6666

R23
router bgp 65002
neighbor 200.99.23.1 remote-as 19999
address-family ipv4
neighbor 200.99.23.1 activate

router ospf 1
default-information originate

R24
ip prefix-list DEF permit 0.0.0.0/0

route-map DEF permit 10
match ip address prefix-list DEF
match tag 19999

router bgp 65002
neighbor 200.99.24.1 remote-as 19999
address-family ipv4
neighbor 200.99.24.1 activate

router ospf 1
default-information originate route-map DEF

R12、R23、R24
ip access-list standard NAT
permit 10.0.0.0 0.255.255.255

R12
interface g0/0
ip nat outside

interface range g0/1-3
ip nat inside

interface loopback 0
ip nat inside

ip nat inside source list NAT interface g0/0 overload

R24

ip nat inside

R23/R24

ip nat outside

ip nat inside

interface loopback 0
