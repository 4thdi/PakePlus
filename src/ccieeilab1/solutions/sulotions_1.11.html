R61\R62\R70
crypto keyring KR vrf WAN 【R70 has the right keyring config, just copy it to R61\r62】
pre-shared-key address 0.0.0.0 0.0.0.0 key cisco

crypto isakmp policy 10
no hash md5
authentication pre-share
group 2

R61
interface GigabitEthernet0/0
vrf forwarding WAN
ip address 100.5.61.2 255.255.255.252
interface Loopback0
vrf forwarding WAN
ip address 10.6.255.61 255.255.255.255

router bgp 65006
address-family ipv4 vrf WAN
network 10.6.255.61 mask 255.255.255.255
neighbor 100.5.61.1 remote-as 10000
neighbor 100.5.61.1 activate

interface Tunnel0 【Check and fix the configuration of the tunnel interface 0 to make it the same as shown below】
ip address 10.200.0.61 255.255.255.0
ip mtu 1440
no ip nhrp map 10.2.255.24 10.200.0.1
ip nhrp map 10.200.0.1 10.2.255.24
ip nhrp map multicast 10.2.255.24
ip nhrp network-id 1010
ip nhrp nhs 10.200.0.1
no ip nhrp redirect
ip nhrp shortcut 【this config will not shown but you need to do it】
tunnel source Loopback0
tunnel mode gre multipoint
tunnel vrf WAN
tunnel protection ipsec profile prof
